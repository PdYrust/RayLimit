# Examples

This section collects practical, copyable command sequences that reflect the product exactly as it exists today.

## How To Use These Examples

Treat each example as a workflow, not just a command list:

1. inspect the runtime
2. preview the change
3. read the observation and decision
4. execute only when the path is concrete
5. verify cleanup separately when you test remove
