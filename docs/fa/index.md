# مستندات فارسی RayLimit

این بخش مسیر فارسیِ شروع کار با RayLimit را فراهم می‌کند.

هدف این بخش، کمک به کاربران فارسی‌زبان برای نصب، اجرای اولیه، و درک سریع رفتار فعلی RayLimit است. برای جزئیات عمیق‌تر فنی، مسیر انگلیسی همچنان مرجع کامل‌تر است.

از انتخاب‌گر زبان در نوار بالای سایت برای جابه‌جایی بین فارسی و انگلیسی استفاده کنید.

## شروع سریع

- [فرمان‌های رایج](getting-started/common-commands.md)
- [استفادهٔ عملی](getting-started/practical-usage.md)

## این بخش چه چیزی را پوشش می‌دهد؟

- تفاوت بین نصب از بستهٔ انتشار و اجرا از سورس
- فرمان‌های رایج برای کشف، inspect، و preview
- مدل dry-run-first و معنی blocked
- مسیر مناسب برای رسیدن سریع به خروجی مفید
